Esse projeto visa ampliar minha base de frontend + manipulação de dados + UX.

Visão do Projeto:

* Entrada:
    * Upload de arquivo (.csv ou .json)
    * OU input manual (textarea)
* Processamento:
    * Converter:
        * CSV to JSON
        * JSON to CSV
* Saída:
    * Mostrar resultado na tela
    * Botão para baixar arquivo convertido
* 100% frontend (client-side) (podendo talvez evoluir para back com node no futuro)
* Tudo roda no navegador
